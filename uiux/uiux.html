    Overview

    • 자립해냄 앱 소개

      ‘한국청소년 상담복지개발원’에서 출시한 위기 청소년의 자립을 돕는 공공복지 앱입니다

    • 선정 이유

      매체 이용에 민감한 청소년들이 주 사용자임에도 앱의 디자인과 정보를 얻는 과정이 불편하고 정확하지 않아 리디자인을 위해 선정했습니다

    • 기여도

      팀프로젝트 (95%)

    Prototype

    시작 화면

    중요도가 높은 카테고리는 위로, 스크롤을 추가해 레이아웃을 여유롭게 배치 페이지를 건너뛰고 바로 가기를 사용함으로써 동작의 복잡함과 사용을 줄일 수 있습니다

    페이지 차별화

    유사 페이지 구분을 위해 일러스트 사용

    쉼터 및 지도 검색

    상세 보기에서 지도로 위치와 기관 정보들을 쉽게 파악

    북마크 추가

    눈여겨 본 기관을 즐겨찾기 등록하여 빠른 이용

    바로 가기 링크

    카테고리를 나열해 페이지 이동을 줄임

    서브 버튼

    공간의 여유를 주는 모션과 기존 앱에서는 여러 카테고리 안으로 숨겨져 있던 긴급전화를 메인 홈의 서브 버튼으로 꺼내 위급한 상황 시 앱을 통하여 사용자가 즉각 도움을 요청할 수 있도록 디자인했습니다

    Problems
      & Solution

    • 문제점

      읽어야 하는 부분과 눌러야 하는 부분이 불명확함

      화면 여백이 모호하고 기기별 화면 크기를 고려하지 않음

      사용자가 지루함을 느끼고 정보를 얻기 어려운 UX 디자인

    • 해결 방법

      스크롤을 추가해 레이아웃을 여유롭게 배치

      선택, 비선택 항목에 대비를 주어 명확하게 구분함

      사용자 편의를 고려해 기능을 추가하고 사용 중 불쾌함을 줄임

    Review

    • 잘한 점

      팀원들의 빠른 피드백 및 피드백 반영

      개인의 시간까지 내어가며 팀 프로젝트 완성도를 높임

      적극적인 의견 제시로 디자인 후보가 많아져 쉽게 결정할 수 있었음

    • 부족한 점

      앱의 아이콘들의 통일성이 부족했다는 점

      런처 아이콘의 디자인이 원본보다 미흡한 점

      프로토타입 과정의 시간이 오래 걸린 점(시간 분배의 아쉬움)

    • 프로젝트 완료 리뷰

      공공기관 앱을 선정하여 경쟁사 및 유사 앱의 장단점 파악이 어려웠습니다. 또한 앱에서 기본적으로 제공하는 정보들이 많아 새로 리디자인 하기도 어려웠습니다. 그래서 반복되는 레이아웃을 줄여보면 어떨지 고민했던 것 같습니다.

      팀원들과 함께 주말 개인 시간까지 나누어가며 노력했기 때문에 아쉬움보다는 뿌듯함이 더 큰 프로젝트였습니다. 초반에 디자인에 자신이 없어 어려웠지만 팀원들의 도움을 많이 받았고 긍정적인 모습들을 보면서 더 잘하고 싶다는 동기부여가 생기는 과정이었습니다.
