    Problems
      & Solution

    • 1. 기존의 video를 활용하였으나 기본으로 고정된 사이즈로 인하여 비율 제어 문제 발생

      유사한 video로 대체하여 iframe 태그를 이용해 비율 제어 문제를 해결함

    • 2. 이미지 hover 시 slide-up 텍스트 박스가 이미지 크기에 맞지 않는 문제 발생

      img 태그에 사용한 이미지 사이즈의 비율이 1 : 1이 아니어서 사이즈 수정 후 해결함

    • 3. 스크롤 위치에 따른 nav 바 변화가 구현되지 않는 문제 발생

      JavaScript marginTop으로 이벤트 발생할 것이라 생각하였지만 해결 불가
      JavaScript transform:'translate(0)'을 적용시켰더니 문제를 해결함
