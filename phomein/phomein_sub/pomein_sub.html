    Problems
      & Solution

    • 1. 양쪽 영역 hover 시 이미지가 같이 움직이는 문제 발생

      각 영역 hover 시 영역의 width 값만 변경되는데 해당 영역에 위치한 이미지가 같이 움직임

      width:auto 와 transform: translate() 값으로 위치 조절 후 해결함

    • 2. 메인의 텍스트 AOS가 실행될 때 백그라운드 gif가 어둡게 실행되지 않는 문제 발생

      gif를 background: url()로 삽입한 후 linear-gradient(), background-blend-mode: multiply로 어두워지도록 하였으나 처음부터 어두운 상태인 문제 발생

      객체의 'load' 이벤트를 기다리는 이벤트 리스너를 설정, 이벤트 리스너 함수 내부에서 setTimeout 함수를 사용, setTimeout 함수는 지정한 시간이 경과한 후 특정 함수를 실행할 수 있게 해주고 setTimeout의 두 번째 인자는 함수가 실행되기 전의 지연 시간을 밀리 초로 실행되도록 설정, 'brightness(0.5)'; 적용하여 요소의 밝기를 조절하여 문제를 해결함
